# RPAのテスト用のダミーサイト (Dummy Product Data Site)

## 概要
本サイトは、Web自動化ツールやRPA（Robotic Process Automation）の動作テスト、およびシナリオ作成のトレーニングを目的として構築されたダミーサイトです。
実在しない架空の製品情報をまとめたデータテーブル（表）が配置されており、Webスクレイピングやデータ抽出アクションのテストを安全に行うことができます。

## 特徴
* **安全なテスト環境**: 掲載されている製品データはすべて架空のものです。誤操作によるデータの破壊や、外部サービスへの影響を気にすることなく自動化のテストが可能です。
* **データテーブルの取得練習**: RPAツールにおける「Webページからデータを抽出する」アクションの対象として最適な、構造化されたHTMLテーブル（`<table>`）を使用しています。
* **シンプルなDOM構造**: 複雑な構成を避け、自動化ツールのセレクター（CSSセレクターやXPathなど）をスムーズに取得できるよう設計されています。

## 想定されるテストシナリオ
本サイトを使用して、RPAツールで以下のようなシナリオのテスト・練習が可能です。

1. **データの一括取得**: Webページ上の表全体を読み込み、データテーブルとしてツール内に格納する。
2. **条件分岐のテスト**: 取得したデータの中から、特定の条件（例：「在庫あり」の製品のみ、価格が「10,000円以上」など）に一致する行だけを抽出する。
3. **ファイル出力**: 取得した製品データを、CSVファイルに書き出す。
4. **UI要素の操作**: （※検索ボックスやボタンを実装した場合）特定のキーワードを入力して検索ボタンをクリックするWebブラウザ操作のテスト。

## ご利用にあたっての注意事項
* **架空のデータです**: 本サイトに掲載されている企業名、製品名、価格などはすべてフィクションであり、実在のものとは一切関係ありません。
* **アクセス頻度について**: テストを実行する際は、サーバーに過度な負荷をかけないよう、ループ処理の中に適切な待機時間（ウェイト）を設定してください。
